Для выполнения автотестов необходимо:
1. Подготовка окружения.
Python 3.14
Созать файл requirements.txt, в котором указаны все необходимые библиотеки и установить зависимость pip install -r requirements.txt
2. Запуск приложения
Установить Docker Desktop (необходимо проверить, чтобы компьютер поддерживал данную программу). В терминае выполнить команду docker-compose up -d
3. Заупск тестов
Для запуска тестов в терминале выполнить команду pytest --alluredir=allure-results
4. Отчёт о тестировании
Для создания отчёта выполнить команду allure serve allure-results
